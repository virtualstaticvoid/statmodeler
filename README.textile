h2. StatModeler

StatModeler is made up of a Ruby DSL for creating statistical, quantitive or mathematical models and an calculation engine for the calculation of models.

The DSL provides a semi-declarative syntax for defining the various aspects of a model, including:
* Source data and mapping
* Model components
* Aggregation and filtering
* Calculations
* Outputing of results

The calculation engine loads the source data, evaluates the model and produces the output.
